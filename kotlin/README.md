## Run A Kotlin Program
```
kotlinc <name of source file> -include-runtime -d hello.jar
```
The extension of a kotlin source file is "kt"

## Execute 
The extension is "jar"

The jar file was created by kotlinc

```
java -jar <jar file> 
```
